# Overview

Open Knowledge Format (OKF) makes Markdown knowledge bundles portable between humans, agents, and tools. Each non-reserved Markdown file is a concept with a required `type` frontmatter field.

# Examples

Use [pgvector](/concepts/pgvector.md) as a derived index while keeping the Markdown bundle authoritative.
