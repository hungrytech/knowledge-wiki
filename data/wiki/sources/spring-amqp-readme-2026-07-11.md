```markdown
Spring AMQP
===========

This project provides support for using Spring and Java with AMQP 0.9.1,
and in particular RabbitMQ.

# Reporting Security Vulnerabilities

Please see our Security policy.

# Checking out and Building

To check out the project from GitHub and build from source using Gradle, do
the following:

    git clone git://github.com/SpringSource/spring-amqp.git
    cd spring-amqp
    ./gradlew build

To build and publish jars to your local Maven repository:

    ./gradlew publishToMavenLocal

To build reference documentation (results will be in `build/site`):

    ./gradlew antora

## Additional Resources

* Spring AMQP Homepage
* Spring AMQP Source
```
