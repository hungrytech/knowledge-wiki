# Overview

This project keeps human-editable Markdown as the source of truth. Hermes curates documents and relationships; the local API and UI make the graph explorable.

# Schema

The derived PostgreSQL catalog projects [Open Knowledge Format](/concepts/okf.md) concepts and their cross-links. [pgvector](/concepts/pgvector.md) is available for a later local embedding pipeline.
